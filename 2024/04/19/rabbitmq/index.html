# 什么是 RabbitMQ？

RabbitMQ 是一个消息中间件 ( Message broker )，用来接受并且发送消息。可以把 RabbitMQ 比作为邮局：当你将一份邮件投递进邮箱时，可以肯定的是这封信最终会被邮递员传递到你指定的收件人 (recipient) 手上。在这个比喻中， RabbitMQ 就是一个邮箱、邮局以及邮递员。 RabbitMQ 和邮局之间的主要的区别就是， RabbitMQ 不能处理纸张，而是接收，存储并且发送二进制的 blob 数据，也就是消息 (message) 。

RabbitMQ 中的一些专有名词：

  • 生产者 (producer)：仅仅发送消息。发送消息的程序就是生产者。
  • 队列 (queue)：队列就时 RabbitMQ 中的邮箱。尽管消息是以流的形式穿过 RabbitMQ 和你的程序，他们仍能被存储在队列中。队列仅受主机内存和磁盘限制的约束，它本质上是一个大的消息缓冲区。一个队列可提供给多个生产者和消费者发送和接收消息。
  • 消费者 (consumer)：消费与接收意思相近的。一个消费者是一个大多数时间等待着接收消息的程序。
